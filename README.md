# MSN Hope Website

This is an editable, mobile-responsive static website for MSN Hope.

## Preview locally

1. Open the `MSN_Hope_Website` folder.
2. Double-click `index.html`.
3. The site will open in a web browser.

## Edit the website

- **Words and links:** Open `index.html` in any text editor.
- **Colors and layout:** Open `styles.css`.
- **Navigation and animations:** Open `script.js`.
- **Photos:** Replace files inside the `assets` folder, keeping the same filenames, or update the image paths in `index.html`.
